Start a Code Club

It’s awesome that you are willing to start a Code Club! Since you’ve already got a venue, you are almost ready to get kids coding. Please read the following steps below to be sure you have everything you need to get started. After reading, feel free to contact us if you have any further questions!

If you are willing to start a Code Club in your local community just follow these easy steps:

Step 1:

Please read the following info. There are 3 essentials to starting a Code Club.

How a code club work:

There are 3 essential elements to starting a Code Club:

  1. Approval from the venue (Library manager, community centre organiser, school Principal etc.)
  2. A volunteer who will run the sessions (the usual commitment is approximately 1 hour per week). It is important that the volunteer holds a valid Working With Children Check. We recommend potential volunteers get their valid Working With Children Check as soon as possible.
  3. Kids! (the average class size is about 15 children, however this is up to the discretion of the volunteer and venue). Kids need to have access to a computer or laptop. They could bring their own or often a computer room works great!

Step 2:

Please register your club on the map via this link to access materials:

Register Your Club

This will give you access to some extras from Code Club, including some door signs and fun bits to help get the kids excited.

Step 3:

Encourage kids from your local area to sign up. We recommend having around 15 in the class to allow for collaboration and be sure there won’t be too much strain on the instructor.

Step 4:

You will need to download Scratch to everyone’s computers in the class – or if they have a live internet connection you can use the web version. Click here to download Scratch.

We recommend that the lessons be read a week ahead to help the instructor prepare. As long as you stay on top of it, Code Club is wonderfully simple and super easy to operate!

NB: If your school decides to start another club, great! Be sure to register every club in your location on the map. This helps us track our impact as we grow.

Thank you for your interest. We hope that the process is clear for you and that you have an easy journey to #getkidscoding! If we can answer any more questions or help you in any way, let us know!
