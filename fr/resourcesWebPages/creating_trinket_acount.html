Créer des comptes Trinket

Comptes Trinket

Trinket peut être utilisé sans un compte. Les projets sont automatiquement sauvegardés en ligne et ils peuvent être accédés à partir d’un lien dans Code Club ou à la maison.

Un compte Trinket permet de retrouver tous vos projets et d’y faire des mises à jour facilement. Sans un compte, vous devrez faire le suivi des liens URL pour les projets individuels et un nouveau lien URL sera créé à chaque fois que le projet sera modifié.

Créer des comptes Trinket

Notez que l’autorisation parentaleest requise afin de créer des comptes pour les enfants de moins de 13 ans. Code Club recommande que les comptes Trinket, si utilisés, soient créez par l’hôte ou les parents. Si l’hôte créer les comptes, il sera plus facile de gérer les noms d’utilisateur et les mots de passe perdus pendant les sessions Code Club mais il aura aussi cette responsabilité en dehors du Code Club. L’hôte peut aussi être le point de contact pour tout incident d’abus de compte.

Créer plusieurs comptes Trinket

Il est possible pour l’hôte de créer plusieurs comptes Trinket avec une seule adresse courriel gmail. Si l’hôte utilise l’adresse courriel codeclubhost@gmail.com, les adresses courriels de la forme codeclubhost+member1@gmail.com peuvent aussi être utilisées pour créer des comptes Trinket.
