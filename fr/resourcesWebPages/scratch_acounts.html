Créer plusieurs comptes Scratch

Les enfants qui ont un compte Scratch peuvent sauvegarder leur projets en ligne pour les accéder à la maison et à l’école.

Quand les enfants sont connectés à leur compte Scratch dans l’éditeur en ligne, leur projets seront automatiquement enregistrés pour éviter la perte de leur travail.

Les projets peuvent être partagés en ligne avec la communauté Scratch ce qui permet aux enfants d’avoir un vrai public pour leur créations.

Il est possible pour l’hôte de créer plusieurs comptes Scratch à partir d’un compte enseignant Scratch.

Les parents peuvent aussi aider les enfants à créer leur propre compte Scratch qui peut être utilisé à la maison et dans les Code Club.

Veuillez noter qu’une autorisation parentale est requise afin de créer un compte Scratch pour les enfants de moins de 13 ans.

Compte enseignant et étudiant Scratch

Un hôte peut utiliser un compte enseignant Scratch afin de créer plusieurs comptes Scratch qui seront gérés par un enseignant ou un autre éducateur. Allouez un délai d’au moins 24 heures pour la création d’un compte enseignant Scratch puisqu’ils sont révisés manuellement.

Si vous utilisez un compte enseignant ou un compte étudiant, Code Club recommande qu’il soit créé par l’hôte puisqu’iI requiert la vérification de l’autorisation parentale, la gestion des réinitialisations de mot de passe, la modération des commentaires et la gestion d’autres problèmes reliés aux comptes.

La page FAQ compte enseignant Scratch explique comment créer un compte enseignant Scratch et une Classe avec des compte étudiant Scratch pour les enfants.

Les comptes enseignant Scratch sont une nouveauté. Présentement, un compte Scratch peut seulement appartenir à une Classe, les enfants qui utilisent Scratch lors des leçons devront utiliser un compte différent de celui utilisé pour leur Code Club. Notez que lorsque l’enseignant termine une Classe Scratch, les enfants ne seront pas capable de se connecter à leur compte mais leurs projets partagés resteront visibles.

Plusieurs comptes Scratch à partir d’une seule adresse courriel

Si les limites des comptes enseignant Scratch posent problème, il est possible de créer plusieurs comptes Scratch avec une seule adresse courriel. Ceci veut dire que l’hôte peut créer des comptes pour les enfants de votre club sans qu’ils n’aient besoin d’avoir leur propre adresse courriel.

Si les enfants créent leurs propres comptes, il pourrait s’avérer utile de partager leur noms d’utilisateur et leur mots de passe avec vous, au cas où ils les oublieraient.

  • Rendez vous sur scratch.mit.edu et cliquez sur ‘Rejoindre Scratch’ dans le coin supérieur droit de l’écran.

  • Choisissez un nom d’utilisateur et un mot de passe comme indiqué. Si vous créez des comptes pour votre Code Club, vous voudrez peut-être planifier un style uniforme. Par exemple:

    • Nom d’utilisateur = les initials de l’enfant + les initials de l’école (ex, SSCStJs)
    • Mot de passe = StJosephs1
Veillez à ne pas inclure d'informations d'identification personnelle dans les noms d'utilisateur. Scratch ne permet pas aux enfants d'utiliser leurs noms réels.
  • Sur la prochaine page, complétez les détails pour l’enfant qui utilisera le compte et cliquez sur suivant.
  • Entrez l’adresse courriel de la personne qui gèrera les comptes des enfants. Cliquez sur suivant.
  • À ce stade-ci, vous recevrez un courriel vous demandant de confirmer votre adresse courriel. Cliquez sur ‘‘OK Allons-y!’’ et vous serez ramené à l’éditeur Scratch, prêt à faire quelque chose de génial!
  • Vous avez réussi! Pour créer un autre compte, déconnectez-vous et répétez la procédure en utilisant la même adresse courriel et différents noms d’utilisateur (vous pouvez réutiliser le même mot de passe si vous le désirez).
Top conseil!

Assurez-vous que l’individu en charge du club tienne un registre des noms d’utilisateur et des mots de passe dans un endroit sécuritaire. Quelques Code Clubs créent des cartes laminées avec le nom d’utilisateur et le mot de passe. Ils les remettent à chaque enfants au début de la session et les reprennent à la fin. Plus aucuns mots de passe oubliés!

Some Code Clubs make a laminated card with the username and password, which they give out to each child at the beginning of the session and collect back in at the end. No more forgotten passwords!

Partager le travail en ligne

Utiliser des compte Scratch signifie que les projets Scratch de chaque utilisateur peuvent être partagés pour le plaisir de la communauté Scratch.

Vous n’avez pas à partager les projets mais souvenez-vous qu’ils peuvent être visionnés par quiconque.

Les utilisateurs ont aussi l’option de commenter les autres projets Scratch. Vous voudrez peut-être éteindre l’option commentaires s’ils partagent leurs projets et leur demander de commenter de manière responsable sur les autres projets du site.

Les enfants devraient être mis au courant des Children should be made aware of the Règles de conduite en communauté de Scratch.
