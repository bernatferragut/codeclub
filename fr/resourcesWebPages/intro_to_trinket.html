Introduction à Trinket

Qu'est Trinket?

En complétant nos projets HTML & CSS et Python, nous recommandons d’utiliser Trinket afin d’écrire et d’exécuter du codage en ligne.

  • Il n’y a aucune application à installer afin d’utiliser Trinket, car il fonctionne uniquement dans un navigateur web.
  • Essayez Trinket dans votre lieu avant de commencer un cours. Si vous avez des problèmes, contactez Trinket Trinket FAQ.

Il est important de comprendre que Trinket a créé des pages web publiques et des projets qui peuvent être visionnées par n’importe qui ayant le lien. Les enfants auront peut-être besoin d’un rappel sur l’importance du partage d’informations personnelles en ligne.

Utiliser Trinket

Trinket est disponible en ligne à trinket.io. Pour les projets Code Club, les enfants obtiendront un url afin d’ouvrir soit un projet libre ou un projet de démarrage contenant les ressources.

Veuillez visiter Intro à HTML & CSS et Intro to Python pour de plus amples informations sur l’utilisation de Trinket pour ces cours.

Sauvegarder les Trinkets

Les comptes Trinket facilitent la trajectoire et les mises à jours des projets. Veuillez noter que la permission parentale est requise pour les comptes Trinket pour les enfants de 13 ans et moins, visiter notre page Trinket Accounts pour de plus amples informations.

Trinket peut aussi être utilisé sans avoir à créer de comptes. Afin de sauvegarder le travail, vous pouvez ‘Partager’ pour recevoir le lien qui peut être utilisé afin de visionner le projet. Veuillez noter qu’il y aura un nouveau lien à chaque changement de projet.

Si les enfants n’ont pas de comptes, nous recommandons de faire le suivi des liens à leurs Trinkets dans un document pour que ce soit facile d’y retourner.
