Introduction à Python

Qu’est-ce que Python?

Python est un langage de programmation basé sur du texte, d’usage général, qui est largement utilisé. Python est fréquemment utilisé en tant que premier langage de programmation basé sur du texte puisqu’il est simple, lisible et vous pouvez compléter vos tâches avec moins de codage que plusieurs autres langages. Python soutient les ‘‘tortues graphiques’’ et les autres modules qui rendent le codage plus amusant et coloré.

Python est utilisé pour de vrai projets dans l’industrie donc il s’agit d’un langage utile à apprendre. Nos projets utilisent la dernière version - Python 3.

Trinket

Nous recommandons l’utilisation de l’éditeur Trinket pour créer et exécuter les projets Python dans un navigateur Web. Trinket offre plusieurs avantages qu’un éditeur de text hors ligne n’offre pas:

  • Les enfants peuvent voir leur codage Python et ses données de sortie côte à côte;
  • Les enfants peuvent sauvegarder leur codage en ligne, ce qui leur permettra d’accéder aux projets à leur club et à la maison;
  • Les enfants peuvent partager leur projets avec les autres et remixer des projets qui leur auront étés partagés.

Le premier projet Python (nommé About Me ) explique en détail la création, la sauvegarde et l’exécution d’un projet simple de Python.
