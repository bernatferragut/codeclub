Écriture HTML et CSS hors connexion

Il y aura peut-être des occasions où votre enfant devra écrire du codage HTML & CSS en mode hors connexion. Ici-bas se trouve un court guide afin d’écrire, de modifier et de visionner HTML & CSS hors connexion.

Écriture

Toutes les ressources des projets téléchargeables incluent les fichiers suivants:

  • Example.html
  • style.css

Assurez-vous que ces fichiers (et chaque image) sont dans les même dossiers. (Si vous désirez écrire votre propre HTML, vous pouvez télécharger les modèles de pages web dans ‘Ressources des projets’ pour chaque projet HTML.)

Ouvrez ces fichiers dans un éditeur de texte. Nous recommandons Sublime Text 3 mais il y a de nombreux bons éditeurs disponibles. La manière la plus facile d’ouvrir les fichiers .html et .css pour vos pages web, est de souligner les deux fichiers et de cliquer avec le bouton droit afin de les ouvrir dans un éditeur de texte.

Sauvegarder

Après avoir écrit votre HTML & CSS, sauvegardez vos fichiers. Assurez-vous que les fichiers aient des extensions de fichier .html et .css.

Visionnement

Afin de visionner une page web, double-cliquez le fichier Example.html pour l’ouvrir dans un navigateur.

Si votre fichier HTML s’ouvre dans un éditeur de texte, assurez-vous qu’il a une extension de fichier .html.

La manière la plus facile de visionner les changements, est d’ouvrir vos fichiers dans un navigateur et dans un éditeur de texte. Vous pouvez ensuite effectuer des changements dans l’éditeur de texte, sauvegarder vos changements et cliquer sur ‘refresh’ dans le navigateur pour voir votre page web mise à jour.
