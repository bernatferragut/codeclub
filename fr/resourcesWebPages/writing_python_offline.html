Python hors connexion

Python offline

Nous recommandons d’utiliser Trinket pour écrire Python en ligne puisqu’il y a de nombreux avantages:

  • Les enfants peuvent sauvegarder des projets dans Trinket et les accéder à l’intérieur et à l’extérieur de leurs sessions Code Club sans avoir besoin de télécharger ou d’installer un logiciel;
  • Les enfants peuvent facilement partager des projets avec d’autres, ce qui leur offre une plateforme pour leur travail. Ils peuvent facilement obtenir des commentaires des autres et offrir des commentaires à leur camarades.

Par contre, vous ne pourrez peut-être pas utiliser Trinket, particulièrement si vous utilisez des ordinateurs sans connexion internet.

Télécharger et installer Python

Vous pouvez utiliser ce lien afin de télécharger la version 3.2 de Python.

Écrire et exécuter un programme simple de Python

L’installation de Python installera aussi l’éditeur IDLE Python qui est la manière la plus simple afin d’écrire et d’exécuter les programmes de Python. IDLE constitut de 2 fenêtres, l’éditeur (où les programmes sont écrits) et le ‘Shell’ (où les programmes sont exécutés).
