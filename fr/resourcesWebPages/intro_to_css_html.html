Introduction à HTML et CSS

Qu’est-ce que HTML?

HyperText Markup Language (HTML) est un langage utilisé pour décrire et structurer les pages web. Les documents HTML peuvent inclure des choses tel que du texte, des images, du son et des vidéos en utilisant des étiquettes pour organiser le contenu. Voici un exemple simple de HTML qui inclut un en-tête < H1 >, un paragraphe de texte < p > et une image < image > 

        
          <html>
          <h1>Here is my cat:</h1>
          <img src="kitten.jpg" width="200">
          <p>He's called 'fluffy' because he is fluffy.</p>
          </html>
Qu’est-ce que CSS?

Cascading Style Sheets (CSS) est un autre langage utilisé afin de formater et de styliser des documents HTML. Par exemple, vous pourriez créer le paragraphe de texte au dessus de bleu avec ce code CSS: 

        
          p {
            color: blue;
          }
Utiliser Trinket

En complétant nos projets HTML & CSS, nous recommandons d’utiliser Trinket afin de créer des pages web en ligne. Les trinkets peuvent être intégrés dans une page web. Essayez de modifier les trinkets ci-dessus.

Trinket offre plusieurs avantages au lieu d’utiliser un modifieur de texte hors ligne:

  • Les enfants peuvent visionner un aperçu en temps réel de leur code HTML & CSS;
  • Les enfants peuvent sauvegarder leur code en ligne, ce qui leur permet d’accéder leurs pages web dans leur club et à la maison;
  • Les enfants peuvent partager leur pages web avec les autres et peuvent remixer les pages web partagées avec eux.
  • Le premier projet HTML & CSS (appelé Happy Birthday) explique la création, la sauvegarde et le visionnement d’une simple page web en détail.

Pour de l’information générale sur l’utilisation de Trinket dans un Code Club, visitez nos pages Trinket Introduction et Trinket Accounts.
