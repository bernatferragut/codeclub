faq

Comment Code Club a-t-il débuté?

Code Club a été fondé en 2012 par Clare Sutcliffe et Linda Sandvik, "Nous partageons la conviction que chaque enfant doit être initié très jeune à la programmation et comprendre combien c'est amusant".

Pour en apprendre davantage sur l’inspiration qui a mené la création de Code Club par la co-fondatrice Clare Sutcliffe, écoutez sa présentation Hacking the future. sur TEDxBrighton.

En 2015, Code Club s’est joint à Raspberry Pi. Voici un article de Philip Colligan, le PDG de Raspberry Pi Foundation: Amener un Code Club dans chaque communauté

Est-ce que je peux facturer une session Code Club?

En bref - non. Un de nos principes les plus importants est de ne pas facturer puisque nous offrons tout notre matériel et nos ressources en ligne gratuitement. De plus, nos bénévoles offrent leur temps gratuitement. Nous n’autorisons pas les clubs à facturer les participants puisque ceci va à l’encontre de nos valeurs.

Êtes-vous une association caritative?

Code Club UK est une association caritative enregistrée avec la fondation Raspberry Pi. Au Canada, Code Club Canada est un projet collaboratif avec Kids Code Jeunesse, qui est une organisation à but non lucratif. Même si nous ne sommes pas une association caritative, tous les profits générés par Code Club Canada retournent directement à notre travail et nos opérations.

Est-ce qu’il y a des obligations légales pour créer un Code Club?

Au Canada, Code Club Canada est un projet collaboratif avec Kids Code Jeunesse, un organisme à but non lucratif. Vous pouvez donc payer le prix réduit de 35$ pour les organismes à but non lucratif en cliquant sur ce lien: sterlingbackcheck.ca/kids-code-jeunesse. Sinon, vous pouvez obtenir une vérification de dossier judiciaire gratuite à la station de police de votre quartier. Nous recommandons de faire la vérification de casier judiciaire renforcée.

Dois-je savoir comment coder pour être bénévole?

Pour devenir bénévole du Code Club, vous n'êtes pas obligé d'être un génie en programmation! Par contre, vous devez avoir certaines habiletés de base afin d'être confiant avec un ordinateur. Si vous vous sentez insécure face à vos compétences techniques, essayez un projet Scratch du Code Club Curriculum et vérifiez vos résultats.

Quel groupe d'âge devrait participer au Code Club?

Le contenu du Code Club a été conçu pour les enfants de 8 à 12 ans. Toutefois, des enfants plus jeunes et plus vieux ont fait partie des Code Clubs avec réussite, donc considérez cet âge limite en tant que ligne directrice. Vous devez tenir compte que certains enfants plus jeunes auront besoin d’une assistance supplémentaire pour lire et travailler à travers les projets. De plus, vous allez peut-être trouver que notre contenu est approprié pour les enfants plus vieux, spécialement s’ils n’ont pas d'expérience antérieure dans la programmation.

Qu’enseigne Code Club?

Nous avons présentement des projets d’une durée de six sessions. Chaque section est d’une durée de 6 semaines, sur une base d’un projet par semaine. Les sections 1 et 2 utilisent Scratch afin d’enseigner les bases de la programmation. Les sections 3 et 4 enseignent les bases du développement Web en utilisant HTML et CSS. Les sections 5 et 6 enseignent Python, et ainsi de suite.

Même si nous concevons et livrons nos projets pour qu’ils s’alignent avec l’année scolaire, cela ne veut pas dire que vous devez gérer votre club de cette manière. Par exemple, vous allez peut-être vouloir commencer votre club à la moitié du terme, ou vous allez peut-être décider de ne pas enseigner Scratch et d’apprendre Python en premier. Vous pouvez utiliser nos projets et gérer votre club comme vous le voulez.

Scratch 1.4... ou Scratch 2?

Tous nos projets sont maintenant mis à jour pour Scratch 2 et ils sont disponibles sur notre page de projets. Si vous utilisez toujours Scratch 1.4, ne vous inquiétez pas, les 6 premiers projets de Scratch sont compatibles, bien que les noms des blocs varient à quelques reprises.

Pourquoi me demandez-vous d’inscrire mon club?

Code Club dépend du financement des organismes, fondations et compagnies pour qu’ils nous aident à créer et fournir des ressources gratuites et du support pour nos bénévoles. Pour faire une demande de financement et pour signaler nos accomplissements, nous devons évaluer notre impact afin de voir ce que l’on fait de bien, et ce que l’on doit faire pour s'améliorer. Nous faisons cela en évaluant le nombre de clubs qui ont été démarrés par des bénévoles. En demandant d’inscrire un club, cela aide à sécuriser le futur de Code Club et nous permet de continuer à enseigner la programmation informatique aux enfants pour les années à venir.

Inscrire les clubs est aussi très important pour créer un environnement sécuritaire pour les jeunes qui participent à Code Club parce que cela nous permet de connaître l’identité du bénévole qui gère le club et de faire les vérifications qui doivent être en accordance avec les règlementations locales.

Comment pouvez-vous vous impliquer en tant que bénévole?

Code Club est mené par la communauté. Nous offrons des projets et du matériel de base. Vous pouvez joindre un club existant ou démarrer votre club et trouver un hôte à un lieu de réception telle qu’une école, une bibliothèque ou un centre communautaire.

Comment ça marche?

La structure de Code Club est flexible et peut être modifiée pour vos besoins. Chaque projet prend une heure à compléter et les clubs sont d’une durée d’une heure à chaque semaine. Vous pouvez gérer un Code Club n’importe où avec des ordinateurs, de l'accès internet et un espace adéquat. Par exemple, dans une école, une bibliothèque ou un centre communautaire. Le bénévole n’a pas besoin d'être un expert en la programmation. Si vous pouvez travailler avec les premiers projets par vous-même, vous allez être à l’aise de les faire avec les enfants.
