LANCEMENT DE CODE CLUB

Date: dimanche le 17 septembre, 2017
Lieu: Bibliothèque Benny

Kids Code Jeunesse est fier d’annoncer le lancement de Code Club à travers le Canada! Nous vous invitons, vous et vos enfants, à participer aux célébrations. Venez vous joindre à nous dimanche le 17 septembre au Benny Fab, situé dans la Bibliothèque Benny, pour un atelier gratuit sur la programmation informatique et démos. La journée débutera avec un atelier Scratch de 2 heures pour enfants, basé sur le livre de Chris Hadfield,The Darkest Dark, qui fera partie de la Semaine de la culture scientifique.

Les activités se poursuivront ensuite avec des démos d’outils informatique pour des familles voulant apprendre à programmer avec l’aide de HTML, Python, Sense HAT, Raspberry Pi et Microbit, provenant tous des Projets Code Club. Notre équipe du Code Club sera aussi présente pour vous aider avec vos questions concernant l’éducation sur les habiletés digitales pour votre enfant au Canada.

Horaire:

11h00-13h00: Atelier Scratch: The Darkest Dark (pour les enfants de 8 à 12) L’enregistrement est requis, l’espace est limité: procurez vos billets ici

13h00-14h00: Démos de matériel informatique du Code Club (pour tous les âges, ouvert à tout le public)

14h00-15h00: Session d’information avec le coordinateur régional de Montréal du Code Club Canada pour des bénévoles et lieux potentiels. Pour plus d’informations consultez notre groupe Meetup

Pour de plus amples informations sur le lancement de Code Club Canada, veuillez contacter: bry@kidscodejeunesse.org Bry LeBlanc, Coordonnatrice des communications

MOONHACK

Date: 15 Août, 2017
Location: Online - across the world

En 2016, #Moonhack a brisé un record mondial en ayant 10,207 enfants australiens qui codaient la même journée. Cette année, Code Club Australia se mondialise en tentant d’avoir plus de 100,000 enfants à travers le monde qui coderont pour Moonhack! Un réseau international de Code Clubs incluant @CodeClubCanada @CodeClubSpain @CodeClubNZ @CodeClubBrasil @CodeClubFrance @CodeClubBD se joignent à l’événement du record mondial de @CodeClubAus Moonhack pour les enfants âgés de 8 à 18 ans! Inscrivez-vous aujourd’hui afin de vous joindre à l’amusement mondial le 15 août. Vous pourrez utiliser les projets multilingues gratuits de Moonhack ou créer votre propre projet basé sur le thème de la lune. Programmez toute la journée et soumettez votre projet à temps afin qu’il fasse partie du record mondial. Apprenez-en plus aujourd’hui afin d’être prêt pour le jour du lancement! Moonhack 2017 sera mis sur pied à partir de la Nouvelle-Zélande, le pays le plus près de la ligne internationale de changement de date. L’événement se propagera ensuite autour du globe pour la journée du 15 août dans chaque pays. Dites-nous d'où vous participerez à Moonhack avec les hashtags #Moonhack #Getkidscoding
