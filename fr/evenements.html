Astro Pi - En ligne

Astro Pi est un concours de codage et de science annuel où du code, écrit par des étudiants, est exécuté à la Station spatiale internationale! Nous voulons inspirer le plus d’étudiants possible à faire de la science spatiale avec du code.

Le défi est ouvert aux écoles et aux Code Clubs de tous les pays ASE (la plupart de l’Europe) incluant le Canada et la Slovénie. Deux missions sont disponibles:

Mission Zéro

Les inscriptions se terminent : 26 novembre
Lieu : En ligne - à travers le monde

  • Une mission non compétitive pour les jeunes de 14 ans et moins.
  • Écrivez un programme simple de Python qui affiche votre message et la température.
  • Votre code est assuré d’être exécuté dans l’espace pendant 30 secondes.

La compétition de cette année vient tout juste de débuter, choisissez votre mission!

LANCEMENT DE CODE CLUB

Date : dimanche le 17 septembre, 2017
Lieu : Bibliothèque Benny

Kids Code Jeunesse est fier d’annoncer le lancement de Code Club à travers le Canada! Nous vous invitons, vous et vos enfants, à participer aux célébrations. Venez vous joindre à nous dimanche le 17 septembre au Benny Fab, situé dans la Bibliothèque Benny, pour un atelier gratuit sur la programmation informatique et démos. La journée débutera avec un atelier Scratch de 2 heures pour enfants, basé sur le livre de Chris Hadfield, The Darkest Dark, qui fera partie de la Semaine de la culture scientifique.

Les activités se poursuivront ensuite avec des démos d’outils informatiques pour des familles voulant apprendre à programmer avec l’aide de HTML, Python, Sense HAT, Raspberry Pi et Microbit, provenant tous des Projets Code Club. Notre équipe du Code Club sera aussi présente pour vous aider avec vos questions concernant l’éducation sur les compétences numériques pour votre enfant.

Horaire :

11h00-13h00: Atelier Scratch: The Darkest Dark (pour les enfants de 8 à 12
13h00-14h00 : Démos de matériel informatique du Code Club (pour tous les âges, ouvert à tout le public)
14h00-15h00 : Sesion d’information avec le coordinateur régional de Montréal du Code Club Canada concernant les bénévoles et lieux potentiels pour partir un Code Club

Pour de plus amples informations sur le lancement de Code Club Canada, veuillez contacter bry@kidscodejeunesse.org Bry LeBlanc, Coordonnatrice des communications.

MOONHACK

Date : 15 Août, 2017
Lieu : En ligne - à travers le monde

En 2016, #Moonhack a brisé un record mondial en ayant 10 207 enfants australiens qui codaient la même journée. Cette année, Code Club Australia se mondialise en tentant d’avoir plus de 100 000 enfants à travers le monde qui coderont pour Moonhack! Un réseau international de Code Clubs incluant @CodeClubCanada @CodeClubSpain @CodeClubNZ @CodeClubBrasil @CodeClubFrance @CodeClubBD se joint à l’événement du record mondial de @CodeClubAus Moonhack pour les enfants âgés de 8 à 18 ans!

Inscrivez-vous aujourd’hui afin de vous joindre à l’amusement mondial le 15 août. Vous pourrez utiliser les projets multilingues gratuits de Moonhack ou créer votre propre projet basé sur le thème de la lune. Programmez toute la journée et soumettez votre projet à temps afin qu’il fasse partie du record mondial. Apprenez-en plus aujourd’hui afin d’être prêt pour le jour du lancement! Moonhack 2017 sera mis sur pied à partir de la Nouvelle-Zélande, le pays le plus près de la ligne internationale de changement de date. L’événement se propagera ensuite autour du globe pour la journée du 15 août dans chaque pays. Dites-nous d'où vous participerez à Moonhack avec les hashtags #Moonhack #Getkidscoding
