Bénévoles

Les bénévoles de Code Club se rendent à des centres communautaires une heure par semaine pour aider les enfants de 8 à 12 ans à apprendre comment développer des jeux, des animations et des sites internet en utilisant les ressources de Code Club.

  • Bénévoles
  • Hôtes de Club
  • Parents

La base

  • Un engagement de 12 semaines minimum
  • Une heure par semaine à un centre communautaire
  • Environ 15 enfants âgés entre 8 et 12 ans par club
  • Aucune expérience en codage n’est requise
  • Aidez les enfants à utiliser des projets Code Club pour créer des jeux, des animations et des sites internet.
  • Faites du bénévolat individuellement ou en équipe avec un collègue ou un ami.

“Gérer un code club et voir les enfants s'illuminer lorsqu'ils les enfants ont du plaisir en programmant, ça c’est une activité très valorisante - cela te rappellera la raison pour laquelle tu aimes coder!”

Kevin Browne, Bénévole - Code Club Hamilton

Pour devenir bénévole avec Code Club vous devrez

  • Être disponible une heure par semaine pour aider à diriger une session Code Club dans une école, une bibliothèque ou un centre communautaire
  • Vous engager à être bénévole pour un minimum d’un semestre (12 semaines)
  • Avoir du temps pour préparer la session de Code Club à chaque semaine. Ceci peut inclure de l’organisation, imprimer des ressources et compléter à l’avance le projet que vous montrerez aux enfants
  • Faire votre contrôle de vérification de dossier judiciaire
  • Être prêt à collaborer avec le propriétaire du lieu où se donne l'atelier pour assurer un déroulement sans tracas
  • Être à l’aise avec les ordinateurs. Vous n’avez pas besoin d’être un génie du codage, la plupart des clubs commencent avec des programmes simples comme Scratch mais vous devez être intéressé à apprendre.

    Veuillez noter que malheureusement nous ne pouvons pas couvrir les dépenses de nos bénévoles. Vous n’avez pas le droit de faire payer les membres de votre club. Code Club offre toujours un accès gratuit aux enfants.

Vos avantages

  • Vous jouez un rôle important en aidant les enfants à développer des compétences qui les aideront à coder et aussi à développer leur pensée informatique, la résolution de problèmes, la planification, le design et la collaboration.
  • Les compétences acquises avec Code Club rendront votre curriculum vitae encore plus attirant.
  • Si vous commencez à apprendre à programmer, Code Club est une façon plaisante de pratiquer vos compétences en codage!

Pourquoi devenir bénévole?

  • La gestion d’un club prend seulement 1 heure de votre temps par semaine. Il est donc facile de commencer à soutenir les jeunes de votre communauté!
  • Aucune expérience en codage n’est requise.
  • Vous acquerrez une belle expérience de bénévolat et de nouvelles compétences.
  • Le programme vous permettra de développer vos propres habiletés de codage.
