Introduction to the 'Robo-Dancers' app

Robo-Dancers

The Robo-Dancers app let’s you create your very own dancing robot!

After selecting a robot and some music, the sliders can be used to change variables for the ]starting angle and amount of movement of various body parts.

Code Mode

‘Code Mode’ can be used to edit the value of variables in a text-based programming environment. Further to this, decisions (if and else statements) can be used to customise the dance during odd and even bars.
